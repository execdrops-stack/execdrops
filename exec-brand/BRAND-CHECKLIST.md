# EXEC Brand Check System

## Purpose
Run EXEC like a live brand, not a static website.

## Check cadence
- Daily when active social posting is happening
- Every other day minimum when quiet
- Weekly review every Sunday night or Monday morning

## Daily / every-other-day check
1. Check Printful / Shopify order status
2. Check whether new sales happened
3. Check Instagram and TikTok post performance
4. Decide the next content move
5. Log what changed

## Weekly review
1. Count posts published
2. Count units sold
3. Count site visits and any conversions
4. Review top-performing angles
5. Decide whether to push:
   - mystery
   - alignment lore
   - humor
   - product clarity
6. Set next week's 3 most important moves

## Core metrics
- total orders
- units sold this week
- revenue this week
- active live posts
- top performing post
- email signups
- traffic to execdrops.com

## Current strategic truth
The story is the main product.
The shirt is the object that lets people participate in the story.
So the brand must be monitored like a media property and a store at the same time.
